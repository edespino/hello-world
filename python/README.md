# Python Hello World

A simple Python program that prints "Hello, Foobar!" with unit tests.

## Files

- `hello.py` — Prints "Hello, Foobar!"
- `test_hello.py` — Tests `hello()` function and script output

## Usage

```bash
python3 hello.py
```

## Testing

```bash
python3 -m pytest test_hello.py
```

Or with unittest:

```bash
python3 test_hello.py
```
