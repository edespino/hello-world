# C Hello World

A simple C program that prints "Hello, Foobar!" with a test harness.

## Files

- `hello.c` — Prints "Hello, Foobar!"
- `Makefile` — Compiles `hello.c` into a `hello` binary
- `test.sh` — Builds, runs, and verifies output

## Usage

```bash
make
./hello
```

## Testing

```bash
bash test.sh
```
